# Bilgi Sistemleri Güvenliği Projesi
"Bu repository, Elektrikli Araç Şarj Sistemlerinin (EVCS) operasyonel güvenliğini tehdit eden potansiyel zafiyetlerin keşfi ve belgelenmesine odaklanmıştır. Özellikle haberleşme protokollerinin (OCPP/ISO 15118) manipülasyonu ve yetkisiz erişim senaryolarını inceleyen proof-of-concept (PoC) kodları içerir."
