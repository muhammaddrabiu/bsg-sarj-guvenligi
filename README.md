# Bilgi Sistemleri Güvenliği Projesi

Bu çalışma, Elektrikli Araç Şarj İstasyonlarının güvenliği üzerine yapılan araştırmalar kapsamında hazırlanmıştır. Özellikle OCPP ve ISO 15118 gibi şarj istasyonlarında kullanılan haberleşme protokollerinde ortaya çıkabilecek güvenlik risklerini incelemeyi amaçlamaktadır.

Repository içerisinde, saldırı yüzeylerinin anlaşılması için çeşitli senaryo taslakları ve PoC denemeleri yer almaktadır. Bu senaryolar; mesaj manipülasyonu, yetkisiz erişim girişimleri ve protokol seviyesindeki zafiyetlerin davranışlarını gözlemlemeye yöneliktir.

Bu depoyu, hem teknik analizlerimi geliştirmek hem de proje ekibine katkı sağlamak amacıyla düzenli olarak güncelliyorum.
